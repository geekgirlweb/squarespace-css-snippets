# Heart Like Icon #
## display heart like icon & text inline
```css
.sqs-simple-like.clicked .like-icon, .sqs-simple-like:hover .like-icon {
	display: inline-block;
}
```

## change the hober & clicked color of heart like icon
```css
.sqs-simple-like.clicked .like-icon, .sqs-simple-like:hover .like-icon {
	background-color: #000;
}
```
